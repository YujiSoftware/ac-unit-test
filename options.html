  1. AtCoderの問題ページを開く
  2. ツールバーの をクリック
  3. エディタに貼り付ける

リリースノート

  • Python
    • `python3 Main.py` で、テストが実行できるようになりました。
      再度、テンプレートから読み込むことで反映されます。
    • Tests can now be run in `python3 Main.py`.
      This will be reflected by reloading the template.
  • Go
    • Go 言語に対応しました。
    • Now supports Go language.

テンプレートの選択

入力している内容を上書きします。

初期設定

問題ページの入出力を読み取るために、許可が必要です。
（許可の対象は atcoder.jp のみです）
  • テストメソッドを挿入する箇所に {{ METHOD }} と記述してください。
  • メソッド名を挿入する箇所に {{ NAME }} と記述してください。
  • 入力例を挿入する箇所に {{ INPUT }} と記述してください。
  • 出力例を挿入する箇所に {{ OUTPUT }} と記述してください。
